## 📑 Projeto em Destaque: Caso DevStore

Este repositório contém uma análise completa de modernização de infraestrutura. A startup **DevStore** sofria com servidores locais instáveis e falta de padronização.

### 🎯 Soluções Implementadas no Relatório:
* **Virtualização:** Isolamento de ambientes críticos para testes.
* **Containerização (Docker):** Garante que o software rode igual em qualquer máquina.
* **Cloud Computing:** Migração para AWS/Azure para escalabilidade infinita.
* **Segurança:** Implementação de Firewalls e controle de acessos.
